# Paradise Nursery

Paradise Nursery is a React web application designed for browsing and purchasing indoor plants. The platform allows users to explore various plant categories, manage their shopping cart using Redux Toolkit, and calculate costs dynamically.

## Features
- **Landing Page**: Welcome screen with company introduction and entry button.
- **Product Catalog**: Categorized indoor plants with options to add items to the cart.
- **Shopping Cart**: Dynamic cart management including quantity adjustment, item removal, and subtotal calculation.
- **State Management**: Built using Redux Toolkit.
